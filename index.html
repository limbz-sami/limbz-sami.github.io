Projects


Predict Human Development Index (HDI)
This project is an attempt to built a machine learning model to predict human development index (HDI) based on different world development indicators in a dataset provided by the world bank. A multilinear regression model was built using five indicators that best described HDI.
Ecological Footprint
2016 global ecological footprint dataset was explored with the goal to determine impact of population, Gross Domestic Product(GDP), and Human Development Index (HDI) on ecological footprint. The relationship between HDI and human demands from the planet was an interesting finding. The result indicates a higher level of HDI results in higher human demands from the planet.
Weather data analysis
To prove it gets hotter as we approach the equator, weather data of more than 500 cities across the world was of varying distance from the equator was visualized and a representative model of weather across world cities was created.
